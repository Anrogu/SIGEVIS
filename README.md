# 🏛️ Sistema de Gestión de Vinculación Social (SIGEVIS)

**SIGEVIS** es una plataforma diseñada para digitalizar, centralizar y transparentar el proceso de registro y publicación de vacantes para **Servicio Social** y **Prácticas Profesionales** en dependencias gubernamentales.

---

## 📋 Descripción del Proyecto
Este sistema permite a las instituciones públicas publicar sus necesidades de apoyo técnico o administrativo, permitiendo que los estudiantes encuentren oportunidades alineadas a su perfil académico, eliminando procesos manuales y burocracia innecesaria.

### ✨ Funcionalidades Clave
* **Gestión de Dependencias:** Registro y validación de instituciones gubernamentales.
* **Publicación de Vacantes:** Formulario estructurado con requisitos, actividades y beneficios.
* **Filtros Inteligentes:** Búsqueda por área de estudio, ubicación y horario.
* **Panel de Control (Dashboard):** Estadísticas para el gobierno sobre el impacto de los programas.
* **Seguimiento de Postulantes:** Control del estatus del estudiante desde la solicitud hasta la liberación.

---
